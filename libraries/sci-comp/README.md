# Sci Comp

sci-comp is a library for the [Datagrok](https://datagrok.ai) platform.
It provides pure TypeScript numerical methods:

* Optimization:
  * single-objective:
    - [Nelder-Mead](https://en.wikipedia.org/wiki/Nelder%E2%80%93Mead_method)
  * multi-objective:
    - [MOEA/D](https://ieeexplore.ieee.org/document/4358754)
