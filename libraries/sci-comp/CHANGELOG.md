# sci-comp changelog

## 0.6.0 (2026-05-06)

NCA (Non-Compartmental Analysis) — Phase 1 MVP core:

* `computeNca` orchestrator covering Cmax, Tmax, AUClast, AUCinf,
  %AUCextrap, λz, t½, CL, Vz
* 3 AUC integration methods (linear, log-linear, linear-up/log-down) ×
  naive Float64 and Neumaier-compensated summation = 6 functions plus
  `aucExtrapolateToInfinity` and the `neumaierSum` helper
* 4 BLQ-handling rules (`set-zero`, `set-half-lloq`, `exclude`,
  `missing`) configurable per phase (`preFirstMeasurable`, `embedded`,
  `afterLast`, `consecutiveAfterLast`) — PKNCA convention
* λz auto best-fit (terminal subset search by adjusted R² with
  PKNCA-style `adj.r.squared.factor` tie-breaking) and manual modes;
  numerically stable centered-sum OLS
* IV bolus c0 back-extrapolation (`estimateC0` / `insertC0`) — faithful
  port of PKNCA `pk.calc.c0` chain (`c0 → logslope → c1 → cmin → set0`)
* Mulberry32 PRNG + `deriveWorkerSeeds` (PKNCA-Variant-A sub-stream
  derivation, ready for Phase 2 bootstrap)
* Validated against PKNCA 0.12.1 on 26 reference profiles across three
  datasets (Theophylline 12 subjects, Indomethacin 6, synthetic rat 8) —
  all Phase 1 parameters within the §9.2 tolerances of
  `docs/nca_development_plan_v2.md`
* Reference fixtures generated by `scripts/generate-nca-fixtures.R`
  (R 4.6 + PKNCA 0.12.1); synthetic rat dataset by
  `scripts/generate-dataset-03-rat-simple.R`
