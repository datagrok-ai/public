## About Datagrok

Datagrok is a browser-native scientific data platform that outperforms competing products by an order of magnitude —
because we made a fundamentally different architectural bet, and a philosophy that prioritizes total value to the customer.
While others ship bloated server-side apps, we push
scientific computation to the client using WebAssembly, WebGPU, and custom-built high-performance data and rendering engines.
The result: millions of data points analyzed and visualized interactively in the browser.
We provide integrated cheminformatics, bioinformatics, and data science capabilities in a single platform —
replacing or augmenting fragmented stacks of dozens of other tools that pharma companies currently stitch together.

Our clients include Novartis, J&J, Daiichi-Sankyo, and other major pharmaceutical companies. We're actively expanding
into new therapeutic areas, enterprise use cases, and AI-powered scientific workflows.
