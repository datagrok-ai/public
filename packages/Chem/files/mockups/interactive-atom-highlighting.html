Interactive atom highlighting — RDKit MinimalLib mockup

Reproduces the Tosco RDKit UGM 2025 atom-annotation example, then adds click-to-select on top. Renders via RDKit_minimal.js → SVG, overlays per-atom notes positioned from the SVG itself, and exposes the same data shape (ISubstruct + atomNotes) the Datagrok cell renderer is being extended to consume.
