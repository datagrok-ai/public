**1. Connection without catalogs (Postgres)**
1. Go to **Browse > Databases**
2. Expand **Postgres**
3. Right-click the `Northwind` connection and select **Browse schema** from the context menu
4. Check that you can interact with the structures presented in schema view as with DB Tables (from its context menu)

**2. Connection with catalogs (MS SQL)**
1. Go to **Browse > Databases**
2. Expand **MS SQL** > **NorthwindTest** > **Catalogs**
3. Right-click a catalog and select **Browse** from the context menu
4. A schema view should open showing all schemas within the selected catalog
5. Check that you can interact with the structures presented in schema view as with DB Tables (from its context menu)
