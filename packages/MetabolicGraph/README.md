# MetabolicGraph

`MetabolicGraph` is a [package](https://datagrok.ai/help/develop/develop#packages) for the [Datagrok](https://datagrok.ai) platform
this package allows the visualization and analysis of metabolic networks.
as its base, there is an [escher.js](https://github.com/escher-js/escher) library that is used to visualize the metabolic networks.
the library is completely rewritten in typescript and is used as a dependency in the package.
