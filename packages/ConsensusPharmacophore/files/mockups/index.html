Option A — Per-PDB summary in grid; full detail in context panel
The grid is replaced with a 5-row aggregation (one row per PDB) with colored count columns (n_hbd, n_hba, n_hyd, n_pi, n_xb, n_total). Selecting a row opens the full per-interaction detail (9 rows for 1XKK) in the Datagrok context panel. The Mol* viewer always shows all 25 chain-F spheres. More polished first impression; higher implementation cost.
