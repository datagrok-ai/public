# Observation Timelines

Timeline visualization of observations across animals and study days.

This view creates a compact visualization where each row represents a unique test and each column represents a study day. Within each cell, a scatter plot shows individual measurement values colored by treatment arm.

Key features:
- Color-coded data points by treatment arm for visual group comparison
- Combined measurements from all findings domains
- Click on individual data points to open the subject profile for that animal and connect the points
- Interactive filtering by trearment arm, domain, test, category, and demographic parameters
- Y-axis renderer shows test names with units

This view is useful for quickly identifying:
- Dose-dependent trends in biomarker responses
- Temporal patterns in findings across study days
- Outlier animals within treatment groups
