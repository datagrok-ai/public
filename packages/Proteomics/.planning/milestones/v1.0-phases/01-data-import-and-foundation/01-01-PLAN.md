  <action>
Replace the stub in `packages/Proteomics/src/parsers/maxquant-parser.ts` with the full parser implementation.

**Exports to create:**
1. `parseMaxQuantText(text: string): DG.DataFrame` -- core parser logic (synchronous)
2. `parseMaxQuant(file: DG.FileInfo): Promise<DG.DataFrame>` -- wrapper that reads file then calls parseMaxQuantText

**Implementation steps:**

1. **Parse TSV:** Use `DG.DataFrame.fromCsv(text, {delimiter: '\t'})` to get the raw DataFrame. Do NOT hand-roll TSV parsing.

2. **Filter rows:** Initialize filter with `raw.filter.init(true)`. For each filter column group, check both naming variants:
   - Contaminant: check columns "Potential contaminant" and "Potential.contaminant"
   - Reverse: check column "Reverse"
   - Only-by-site: check columns "Only identified by site" and "Only.identified.by.site"
   - For each, iterate rows and set `df.filter.set(i, false)` where value is "+"
   - Also check Protein IDs column (try "Protein IDs" then "Majority protein IDs"): filter rows where ID starts with "CON__" or "REV__"
   - Clone filtered: `const df = raw.clone(raw.filter);`
   - Set `df.name = 'proteinGroups';`

3. **Parse semicolon-delimited fields:** For Protein IDs and Gene Names columns, extract the primary (first) entry. Add new columns "Primary Protein ID" and "Primary Gene Name" containing the first semicolon-separated value. Keep the original columns intact.

4. **Assign semantic types:**
   - `df.col('Protein IDs')?.semType` or `df.col('Majority protein IDs')?.semType` = SEMTYPE.PROTEIN_ID
   - Also assign SEMTYPE.PROTEIN_ID to "Primary Protein ID" column
   - `df.col('Gene names')?.semType` or `df.col('Gene name')?.semType` = SEMTYPE.GENE_SYMBOL
   - Also assign SEMTYPE.GENE_SYMBOL to "Primary Gene Name" column

5. **Detect and log2-transform intensity columns:**
   - Define prefixes: `['lfq intensity', 'ibaq', 'reporter intensity', 'intensity']` (check in this order, case-insensitive)
   - For each column in df: if it's FLOAT or INT and its lowercased name starts with a prefix, it's an intensity column
   - IMPORTANT: "intensity" prefix must be checked LAST because it's a substring of "lfq intensity" -- the order matters for correct type detection but not for column matching
   - For each intensity column, assign `col.semType = SEMTYPE.INTENSITY`
   - Add new column `df.columns.addNewFloat('log2(' + col.name + ')')`
   - Iterate rows: if `col.isNone(i)` set `DG.FLOAT_NULL`; else if `val > 0` set `Math.log2(val)`; else set `DG.FLOAT_NULL`
   - Assign `log2Col.semType = SEMTYPE.INTENSITY` to each new log2 column

6. **parseMaxQuant wrapper:** Read file content via `await file.readAsString()`, then call `parseMaxQuantText(text)`.

**Key pitfalls to avoid:**
- Do NOT use JavaScript `NaN` for null values in float columns; use `DG.FLOAT_NULL`
- Do NOT forget to check `col.isNone(i)` BEFORE calling `col.get(i)` for intensity values
- Do NOT modify the "Protein IDs" column in place -- create a separate "Primary Protein ID" column
- Use `import {SEMTYPE} from '../utils/proteomics-types';` for semantic type constants
