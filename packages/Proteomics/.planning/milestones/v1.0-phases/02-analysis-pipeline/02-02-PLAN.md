**Helper: getGroupValues(df, colNames, rowIdx):**
Extract non-null float values across the named columns for a single row. For each column name, get the column, check `col.isNone(rowIdx)`, push value if present. Return `number[]`.

**runDifferentialExpression(df, group1Cols, group2Cols, group1Name, group2Name, fcThreshold = 1.0, pThreshold = 0.05):**

1. Add four new columns:
   - `df.columns.addNewFloat('log2FC')` -- fold change
   - `df.columns.addNewFloat('p-value')` -- raw p-value
   - `df.columns.addNewFloat('adj.p-value')` -- BH-adjusted p-value
   - Column name for log2FC should be descriptive: `'log2FC (' + group2Name + ' vs ' + group1Name + ')'`. But keep it simple as `'log2FC'` for downstream compatibility.

2. Per-protein loop (i = 0 to df.rowCount):
   - Extract `vals1 = getGroupValues(df, group1Cols, i)` and `vals2 = getGroupValues(df, group2Cols, i)`
   - If either group has < 2 values: set log2FC, p-value to `DG.FLOAT_NULL` (cannot test). Continue.
   - Compute `log2FC = mean(vals2) - mean(vals1)` (treatment minus control)
   - Call `tTest(vals1, vals2)` -- this returns `{'p-value': number}`. Extract the p-value.
   - Store log2FC and p-value in respective columns.
   - Track testable indices and p-values for FDR correction.

3. BH FDR correction:
   - Build `Float32Array` from testable p-values.
   - Call `fdrcorrection(new Float32Array(pValues), 0.05, 'i')` -- 'i' = Benjamini-Hochberg.
   - Write corrected p-values back to adj.p-value column at testable indices.
   - Set untestable proteins' adj.p-value to `DG.FLOAT_NULL`.

4. Add boolean significant column:
   - `df.columns.addNewBool('significant')`
   - For each row: `significant = !isNone(adjPVal) && Math.abs(log2fc) >= fcThreshold && adjPVal <= pThreshold`

5. Assign semantic types:
   - `log2fcCol.semType = SEMTYPE.LOG2FC`
   - `pValCol.semType = SEMTYPE.P_VALUE`
   - `adjPValCol.semType = SEMTYPE.P_VALUE`

6. Set state tag: `df.setTag('proteomics.de_complete', 'true')`
7. Call `df.fireValuesChanged()`.
8. Return an object `{tested: number, untestable: number}` for the info balloon.

**showDEDialog(df):**
1. Prerequisite checks:
   - `getGroups(df)` -- if null, `grok.shell.warning('Please annotate experimental groups first (Proteomics | Annotate Experiment)')` and return.
   - `df.getTag('proteomics.de_complete') === 'true'` -- if so, `grok.shell.warning('Differential expression already performed')` and return.
2. Get groups from tags. Build dialog with:
   - Info text showing group assignments (read-only): "Group 1: ControlName (N samples), Group 2: TreatmentName (M samples)"
   - `ui.input.float('|log2FC| threshold', {value: 1.0})` with tooltip "Minimum absolute fold change for significance"
   - `ui.input.float('Adj. p-value threshold', {value: 0.05})` with tooltip "Maximum adjusted p-value for significance"
3. On OK: call `runDifferentialExpression(df, group1Cols, group2Cols, group1Name, group2Name, fcThreshold, pThreshold)`.
4. Show `grok.shell.info('DE complete: N proteins tested, M untestable (insufficient replicates)')`.

Export both `runDifferentialExpression` and `showDEDialog`.
