**normalization.ts -- Replace existing stub entirely:**

`medianNormalize(df, colNames: string[])`: For each column name, get the column, compute `col.stats.med`, skip if NaN, subtract median from each non-null value via `col.set(i, col.get(i) - median)`. After all columns, call `df.fireValuesChanged()`. Set tag `df.setTag('proteomics.normalized', 'true')`. Return void (not DataFrame -- in-place modification per user decision).

`showNormalizationDialog(df)`:
- Check `df.getTag('proteomics.normalized') === 'true'` -- if so, `grok.shell.warning('Data already normalized')` and return.
- Build `log2Cols` = columns with `semType === SEMTYPE.INTENSITY` and name starts with `'log2('`.
- Show dialog with `ui.input.columns('Intensity columns', {table: df, available: log2Cols, value: log2Cols})` (all pre-selected).
- On OK: call `medianNormalize(df, colNames)`, show `grok.shell.info('Normalized N columns (median centering)')`.

**imputation.ts -- Replace existing stub entirely:**

`randomNormal(mean, stdev)`: Box-Muller transform: `Math.sqrt(-2 * Math.log(Math.random())) * Math.cos(2 * Math.PI * Math.random())` scaled by mean + stdev * z.

`imputeMinProb(df, colNames: string[], downshift = 1.8, width = 0.3)`: For each column, compute `col.stats.avg` and `col.stats.stdev`. Skip if NaN or stdev === 0. Imputation distribution: mean = avg - downshift * stdev, sd = width * stdev. For each null value (`col.isNone(i)`), set to `randomNormal(imputeMean, imputeSd)`. After all columns, `df.fireValuesChanged()`. Set tag `df.setTag('proteomics.imputed', 'true')`. Return count of imputed values. Signature: `(df, colNames, downshift?, width?) => number`. Return void -- change to return number for info balloon.

`showImputationDialog(df)`:
- Check `df.getTag('proteomics.imputed') === 'true'` -- if so, `grok.shell.warning('Missing values already imputed')` and return.
- Build `log2Cols` as above.
- Show dialog with:
  - `ui.input.columns('Intensity columns', {table: df, available: log2Cols, value: log2Cols})`
  - `ui.input.float('Downshift', {value: 1.8})` with tooltip "Standard deviations to shift left from mean (Perseus default: 1.8)"
  - `ui.input.float('Width', {value: 0.3})` with tooltip "Fraction of standard deviation for imputation distribution (Perseus default: 0.3)"
- On OK: call `imputeMinProb(df, colNames, downshift, width)`, show `grok.shell.info('Imputed N missing values across M columns')`.
