**Test categories and cases:**

**category('Experiment Setup'):**

1. **'setGroups persists group assignments as tag'**: Create a DataFrame, call setGroups with known GroupAssignment, verify `df.getTag('proteomics.groups')` is valid JSON matching the input.

2. **'getGroups returns null for untagged DataFrame'**: Create a fresh DataFrame, verify getGroups returns null.

3. **'getGroups round-trips GroupAssignment correctly'**: Set groups, get groups, verify all fields match (name, columns arrays).

**category('Normalization'):**

4. **'median normalization shifts column medians to zero'**: Create a DataFrame with a log2 intensity column containing known values (e.g., [2, 4, 6, 8, 10] -- median 6). After medianNormalize, verify the column values are shifted by -6 (i.e., [-4, -2, 0, 2, 4]). Check that the new median is approximately 0 (within floating point tolerance).

5. **'median normalization skips null values'**: Create a column with some `DG.FLOAT_NULL` values mixed in. Verify nulls remain null after normalization, and non-null values are shifted correctly (median computed from non-null values only).

6. **'normalization sets proteomics.normalized tag'**: After medianNormalize, verify `df.getTag('proteomics.normalized') === 'true'`.

**category('Imputation'):**

7. **'MinProb imputation fills all NaN values'**: Create a column with some `DG.FLOAT_NULL` values. After imputeMinProb, verify no `col.isNone(i)` remains true for any index.

8. **'imputed values are below the observed mean'**: Create a column with known values (e.g., [10, 12, 14, 16]) and several nulls. After imputation with default downshift=1.8, verify all imputed values are less than the observed mean (13). This confirms the downshift works correctly.

9. **'imputation sets proteomics.imputed tag'**: After imputeMinProb, verify `df.getTag('proteomics.imputed') === 'true'`.

**category('Differential Expression'):**

10. **'DE produces correct log2FC sign for known groups'**: Create a DataFrame with 5 proteins. Group 1 columns have values around 5.0, Group 2 columns have values around 10.0 (clearly higher). Run DE. Verify log2FC column values are positive (group2 > group1 means positive FC).

11. **'DE produces valid p-values between 0 and 1'**: Using the same test data, verify all p-value and adj.p-value entries are between 0 and 1 (or null for untestable).

12. **'proteins with <2 replicates get null p-values'**: Create a protein row where group1 has 1 non-null value and group2 has 3. Verify that protein's p-value is null (DG.FLOAT_NULL / isNone).

13. **'significant column marks proteins passing both thresholds'**: Use test data with one clearly different protein (large FC, small p) and one similar protein (small FC, large p). Verify the significant column is true for the different protein and false for the similar one.

14. **'DE sets proteomics.de_complete tag'**: After runDE, verify `df.getTag('proteomics.de_complete') === 'true'`.

15. **'DE assigns correct semantic types'**: Verify log2FC column has semType Proteomics-Log2FC, p-value and adj.p-value columns have Proteomics-PValue.

```typescript
function makeTestDf(nProteins: number, group1Names: string[], group2Names: string[]): DG.DataFrame {
  const cols: DG.Column[] = [];
  cols.push(DG.Column.fromStrings('Protein ID', Array.from({length: nProteins}, (_, i) => `P${i}`)));
  for (const name of [...group1Names, ...group2Names]) {
    const values = new Float32Array(nProteins);
    // Fill with random-ish but deterministic values
    cols.push(DG.Column.fromFloat32Array(name, values));
  }
  return DG.DataFrame.fromColumns(cols);
}
```
