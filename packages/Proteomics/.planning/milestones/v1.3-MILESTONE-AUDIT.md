### Stale tests fixed during the debt-clearance pass

- **14-03 `filtersScopingNoFlags` + `filtersScopingFallbackToProteinIdWhenNoDisplayName`** — both asserted column membership via `filters.getOptions().look.filters[].column`. Phase 13 round-3 fix `e527d07ba1` discovered the platform serializer strips both `look.filters[]` and `look.columnNames` regardless of which shape is fed in; the columnNames-via-look-config path makes the viewer dock visually but is NOT observable via `getOptions()`. Tightened both to assert only the docking decision (`dockComparisonFilterIfMultiContrast` returns `true` + `findFiltersViewer(tv) !== null`); the visual-membership and Flags-exclusion contracts are pinned by UAT Tests 2 + 4 in this round.
- **14-04 `uniprotPanelPerGroupBarsRender`** — `querySelectorAll('div')` matched both leaf `divText` children AND their parent `divV` container (whose `textContent` concatenates the children); count came back 3 instead of 2. Tightened selector to leaf-only (`d.querySelector(':scope > div') === null`).
