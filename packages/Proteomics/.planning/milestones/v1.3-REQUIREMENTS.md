# Requirements: Proteomics v1.3

**Defined:** 2026-03-06
**Core Value:** Scientists can import proteomics data and go from raw protein quantification to biological interpretation -- all within Datagrok.

## v1.3 Requirements

Requirements for v1.3 Richer Dialogs & UX. Each maps to roadmap phases.

### Spectronaut Parser

- [x] **SPEC-01**: User can import Spectronaut long-format TSV with auto-detection and long-to-wide pivot
- [x] **SPEC-02**: Spectronaut parser filters decoy proteins and applies q-value threshold
- [x] **SPEC-03**: Spectronaut parser extracts R.Condition/R.Replicate for auto-group annotation
- [x] **SPEC-04**: Spectronaut data is tagged as potentially pre-normalized for downstream warning

### Normalization

- [x] **NORM-01**: User can normalize with quantile normalization (client-side TypeScript)
- [x] **NORM-02**: User can normalize with VSN (R script, reads raw intensity columns, fallback to quantile)
- [x] **NORM-03**: User can select normalization method from a dialog with before/after distribution plots
- [x] **NORM-04**: Normalization dialog warns if Spectronaut data is detected as potentially pre-normalized

### Imputation

- [x] **IMP-01**: User can impute with kNN (client-side TypeScript, k=10 default, progress indicator)
- [x] **IMP-02**: User can impute with zero, mean, or median methods
- [x] **IMP-03**: User can select imputation method from a dialog with conditional parameters per method
- [x] **IMP-04**: User can filter proteins by minimum valid values before imputation

### Differential Expression

- [x] **DE-01**: User can select comparison direction from auto-generated group pairs
- [x] **DE-02**: DE dialog shows/hides method-specific parameters based on selected method

### UX Polish

- [x] **UX-01**: Volcano, PCA, and heatmap viewers display descriptive titles
- [x] **UX-02**: DataFrame retains the imported filename as its name

## Out of Scope

| Feature | Reason |
|---------|--------|
| Raw instrument file processing (.raw, .mzML) | Search engines handle this |
| De novo peptide sequencing | Specialized tools exist |
| Spectronaut peptide-level quantification | Doubles parser complexity for marginal gain |
| Tail-Robust Quantile Normalization | Niche improvement over standard quantile |
| Batch effect correction (ComBat) | Different statistical procedure requiring batch metadata |
| Multi-group DE (ANOVA-style) | Fundamentally different statistical tests |
| Web Worker offloading for kNN | Premature optimization; profile first in v1.3 |
