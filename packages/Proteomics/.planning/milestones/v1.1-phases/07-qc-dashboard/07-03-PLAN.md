<objective>
Fix 4 UAT failures in the QC dashboard: circular JSON error on open, missing heatmap, missing bar chart, missing box plot.

Purpose: All failures share one root cause -- auxiliary DataFrames (missDf, barDf, longDf) are registered via grok.shell.addTable() which triggers circular JSON serialization, and tv.addViewer() ignores the dataFrame option so viewers bind to the wrong DataFrame. The fix replaces grok.shell.addTable() + tv.addViewer() with DG.Viewer static factory methods that correctly bind viewers to their auxiliary DataFrames without registering them as top-level tables.

Output: A working QC dashboard where all 7 viewers render correctly when opened from the menu.
</objective>
