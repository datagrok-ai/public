**1. Update package.ts:**

Add import at top: `import {openQcDashboard} from './viewers/qc-dashboard';`

Add new menu entry method in `PackageFunctions` class, positioned after `showPcaPlot` and before `showAllVisualizations`:

```typescript
@grok.decorators.func({'top-menu': 'Proteomics | Visualize | QC Dashboard...'})
static async showQcDashboard(): Promise<void> {
  const tv = grok.shell.tv;
  const df = tv?.dataFrame;
  if (!df) { grok.shell.warning('No table open'); return; }
  openQcDashboard(df);
}
```

This follows the exact same pattern as showPcaPlot -- no DE prerequisite, but group annotations are checked inside openQcDashboard itself.

**2. Create tests/qc-dashboard.ts:**

Create unit tests for the QC computation functions. These test the pure computation logic, not the viewer creation (which requires a running Datagrok instance).

```typescript
import * as DG from 'datagrok-api/dg';
import {category, test, expect} from '@datagrok-libraries/test/src/test';
import {computeMA, computeCV, computeLoessTrend, createMissingnessMatrix, unpivotIntensities, computeMissingBarData, getIntensityColumns} from '../viewers/qc-computations';
import {GroupAssignment} from '../analysis/experiment-setup';
import {SEMTYPE} from '../utils/proteomics-types';
```

Use `category('QC Dashboard', () => { ... })` to group all tests.

Test cases to implement:

**`test('MA computation')`:**
- Create a DataFrame with 3 rows and 4 log2 intensity columns (2 per group).
- Set known values: row 0: [10, 12, 8, 6], row 1: [5, 7, 3, 5], row 2: all null in group2.
- Set semType to SEMTYPE.INTENSITY and prefix names with `log2(`.
- Create GroupAssignment with group1 = first 2 cols, group2 = last 2 cols.
- Call computeMA.
- Assert: row 0 M = mean([8,6]) - mean([10,12]) = 7 - 11 = -4. A = (11 + 7) / 2 = 9.
- Assert: row 2 M and A are null (DG.FLOAT_NULL check via `col.isNone(2)`).

**`test('CV computation')`:**
- Create DataFrame with 2 rows, 3 log2 intensity columns for one group.
- Set known values: row 0: [2, 2, 2] (identical -> CV = 0), row 1: values with known CV.
- Call computeCV.
- Assert: row 0 CV = 0 (identical values -> sd=0, CV=0). Mean = 4 (2^2).
- Assert: row 1 CV > 0 and matches expected value.

**`test('Loess/moving-average trend')`:**
- Create a DataFrame with M and A columns (as if computeMA has already run). Use 20+ rows with A values spread across a range (e.g., 1 to 20) and M values that have a clear pattern (e.g., M = 0.5 * sin(A/5) + noise).
- Call computeLoessTrend with a large window fraction (e.g., 0.5) so the trend is heavily smoothed.
- Assert: `MA_trend` column exists in the DataFrame.
- Assert: `MA_trend` values are not all DG.FLOAT_NULL (trend was computed).
- Assert: the trend values are smoother than the raw M values -- compute variance of MA_trend and variance of M, assert variance(MA_trend) < variance(M).

**`test('Missingness matrix')`:**
- Create DataFrame with 2 rows, 2 intensity columns.
- Set row 0 col 0 = 5.0, row 0 col 1 = null, row 1 col 0 = null, row 1 col 1 = 3.0.
- Call createMissingnessMatrix.
- Assert: result has 2 rows. Row 0: [1, 0]. Row 1: [0, 1].
- Assert: result name is 'Missing Values'.

**`test('Unpivot intensities')`:**
- Create DataFrame with 2 rows and 2 intensity columns, plus protein ID column.
- Set values (some null to test skipping).
- Call unpivotIntensities.
- Assert: result has columns ProteinId, Sample, Intensity.
- Assert: row count equals number of non-null values.
- Assert: result name is 'Intensity Distributions'.

**`test('Missing bar data')`:**
- Create DataFrame with 3 rows and 2 intensity columns (1 per group).
- Set 1 null in col 0 (33.3% missing), 0 null in col 1 (0% missing).
- Call computeMissingBarData.
- Assert: result has 2 rows (one per sample). MissingPct for col 0 ~ 33.3. MissingPct for col 1 = 0.
- Assert: Group column matches group assignment.

**`test('getIntensityColumns')`:**
- Create DataFrame with columns: 'log2(sample1)' with semType INTENSITY, 'other' with no semType, 'log2(sample2)' with semType INTENSITY.
- Call getIntensityColumns.
- Assert: returns ['log2(sample1)', 'log2(sample2)'].
