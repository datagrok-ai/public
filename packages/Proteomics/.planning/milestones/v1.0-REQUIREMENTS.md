# Requirements: Proteomics Package

**Defined:** 2026-02-28
**Core Value:** Scientists can import MaxQuant results and go from raw protein quantification to differential expression results with interactive visualizations — all within Datagrok.

## v1 Requirements

Requirements for initial release. Each maps to roadmap phases.

### Data Import

- [x] **IMPORT-01**: User can import MaxQuant proteinGroups.txt with automatic contaminant and reverse hit filtering
- [x] **IMPORT-02**: Parser auto-detects intensity column type (LFQ, iBAQ, Reporter) and assigns semantic types
- [x] **IMPORT-03**: Parser applies log2 transformation to intensity columns on import
- [x] **IMPORT-04**: Parser assigns semantic types to protein ID, gene name, log2FC, p-value, and intensity columns
- [x] **IMPORT-05**: Package includes a bundled public MaxQuant demo dataset for testing and demonstrations
- [x] ~~**IMPORT-06**~~: *Moved to v2 backlog*

### Experiment Setup

- [x] **SETUP-01**: User can assign sample columns to experimental groups via annotation dialog
- [x] **SETUP-02**: Group assignments persist as DataFrame metadata for downstream analysis

### Analysis

- [x] **ANLY-01**: User can normalize intensity columns using median centering (client-side TypeScript)
- [x] **ANLY-02**: User can impute missing values using MinProb method with configurable parameters (per-sample)
- [x] **ANLY-03**: User can run differential expression (Welch's t-test + BH FDR) between two groups (client-side TypeScript)
- [x] **ANLY-04**: User can run DEqMS differential expression (peptide-count-weighted) via R script
- [x] **ANLY-05**: DE results include log2FC, p-value, and adjusted p-value columns with correct semantic types

### Visualization

- [x] **VIZ-01**: User can open a volcano plot (scatter plot configured with log2FC vs -log10 adjusted p-value, formula lines for significance thresholds)
- [x] **VIZ-02**: User can open a heatmap with proteomics-appropriate defaults (hierarchical clustering, sample grouping)
- [x] **VIZ-03**: User can open a PCA plot colored by experimental group

### Annotation

- [x] **ANNOT-01**: User can click a protein ID in the grid and see UniProt details (name, function, GO terms) in the context panel
- [x] **ANNOT-02**: UniProt info panel triggers automatically for columns with Proteomics:ProteinId semantic type

## Out of Scope

| Feature | Reason |
|---------|--------|
| Raw instrument file processing (.raw, .mzML) | Search engines handle this; not downstream analysis |
| De novo peptide sequencing | Specialized tools exist (PEAKS, DeepNovo) |
| Custom volcano JsViewer | Datagrok's scatter plot handles this natively with formula lines |
| Single-cell proteomics workflows | Future milestone; requires specialized data types |
| Spatial proteomics visualization | Future milestone; requires tissue image overlay |
| Multi-omics integration | Future milestone; cross-domain analysis |
| ProteomeXchange/PRIDE browser | Future milestone; public repository integration |
| Instrument control or LIMS | Out of scope for an analysis package |
