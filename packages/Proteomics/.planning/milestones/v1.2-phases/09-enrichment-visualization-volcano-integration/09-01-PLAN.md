    - wireEnrichmentToVolcano(enrichDf, proteinDf): when enrichDf.currentRowIdx changes, parses Intersection column (comma-separated, trimmed), finds matching rows in proteinDf by gene symbol column, sets proteinDf.selection bits, calls fireChanged(). Returns rxjs.Subscription.
    - wireEnrichmentToVolcano returns Subscription.EMPTY if no gene symbol column found on proteinDf
    - wireEnrichmentToVolcano clears selection (setAll false) before setting new bits
    - openEnrichmentVisualization(enrichDf, proteinDf): creates top-N clone, dot plot, bar chart, docks them into enrichment table view (dot plot RIGHT, bar chart DOWN from dot plot), wires cross-DF selection
  </behavior>
  <action>
    Create `packages/Proteomics/src/viewers/enrichment-viewers.ts` with these exports:

    1. `createTopNEnrichmentDf(enrichDf: DG.DataFrame, topN: number = 15): DG.DataFrame`
       - Get FDR column, build indexed array of {idx, fdr} skipping nulls
       - Sort ascending by FDR, take top min(topN, length)
       - Create BitSet mask, clone enrichDf with mask
       - Truncate Term Name values to 50 chars (append "..." if truncated)
       - Add -log10(FDR) float column: fdr > 0 ? -Math.log10(fdr) : 10
       - Set name to `Top ${count} Enriched Terms`
       - Return cloned DataFrame

    2. `createEnrichmentDotPlot(topDf: DG.DataFrame): DG.ScatterPlotViewer`
       - Use DG.Viewer.scatterPlot(topDf, { x: 'Gene Ratio', y: 'Term Name', sizeColumnName: 'Gene Count', colorColumnName: '-log10(FDR)', markerMinSize: 5, markerMaxSize: 25, title: 'Enrichment Dot Plot' })
       - Return the viewer

    3. `createEnrichmentBarChart(topDf: DG.DataFrame): DG.Viewer`
       - Use DG.Viewer.barChart(topDf, { splitColumnName: 'Term Name', valueColumnName: '-log10(FDR)', valueAggrType: 'avg', barSortType: 'by value', barSortOrder: 'desc', orientation: 'Horizontal', title: 'Top Enriched Terms' } as any)
       - Return the viewer

    4. `wireEnrichmentToVolcano(enrichDf: DG.DataFrame, proteinDf: DG.DataFrame): rxjs.Subscription`
       - Import rxjs from 'rxjs'
       - Find gene symbol column on proteinDf using findColumn(proteinDf, SEMTYPE.GENE_SYMBOL, ['gene name', 'gene symbol'])
       - If not found, return rxjs.Subscription.EMPTY
       - Build Map<string, number[]> gene-to-row index from gene symbol column (trim values)
       - Subscribe to enrichDf.onCurrentRowChanged:
         - Check currentRowIdx >= 0
         - Read Intersection column value
         - Split by comma, trim, filter empty
         - proteinDf.selection.setAll(false, false)
         - For each gene, find matching rows and set selection bits (false for notify)
         - Call proteinDf.selection.fireChanged()
       - Return the subscription

    5. `openEnrichmentVisualization(enrichDf: DG.DataFrame, proteinDf: DG.DataFrame): void`
       - Call createTopNEnrichmentDf(enrichDf, 15)
       - Find or create TableView: grok.shell.getTableView(enrichDf.name) ?? grok.shell.addTableView(enrichDf)
       - Create dot plot and bar chart from top-N clone
       - Dock: dotNode = tv.dockManager.dock(dotPlot, DG.DOCK_TYPE.RIGHT, null, 'Dot Plot', 0.5)
       - Dock: tv.dockManager.dock(barChart, DG.DOCK_TYPE.DOWN, dotNode, 'Bar Chart', 0.5)
       - Wire cross-DF: wireEnrichmentToVolcano(enrichDf, proteinDf)
       - Also wire top-N clone: wireEnrichmentToVolcano(topDf, proteinDf) so dot plot/bar chart clicks also highlight
       - Store subscriptions in a module-level variable; unsubscribe previous before creating new

    Imports: datagrok-api/grok, datagrok-api/dg, rxjs, findColumn from utils/column-detection, SEMTYPE from utils/proteomics-types.
