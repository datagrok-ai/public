**kNN imputation** -- add `imputeKnn(df: DG.DataFrame, colNames: string[], k?: number): number` to imputation.ts:
1. Default k=10
2. Build Float64Array matrix from columns for fast access
3. Build boolean missing mask per row
4. Compute column means as fallback values
5. For each row with missing values (but not all-missing):
   a. Compute Euclidean distances to all other rows on shared non-missing columns
   b. Normalize by dividing by sqrt(sharedCount) for comparable distances
   c. Sort distances, take top k neighbors (or fewer if < k available)
   d. For each missing column in this row: average the non-missing values from neighbors for that column
   e. If no neighbor has a value for that column: use column mean
   f. Set the imputed value via col.set()
6. Show progress via `DG.TaskBarProgressIndicator.create('kNN imputation...')` with percentage updates per row
7. Call `pi.close()`, `df.fireValuesChanged()`, `df.setTag('proteomics.imputed', 'true')`
8. Return total imputed count

Use uniform weighting (average, not inverse-distance weighted) per Claude's discretion -- simpler and standard for MNAR proteomics data.

**Simple imputation methods** -- add to imputation.ts:
- `imputeZero(df: DG.DataFrame, colNames: string[]): number` -- replace nulls with 0
