## v1.1 Requirements

Requirements for v1.1 Parser & QC milestone (Phases 6-7).

### Import

- [x] **IMPORT-01**: User can import a generic CSV/TSV matrix file via file handler
- [x] **IMPORT-02**: User sees a column mapping dialog to select protein ID column and intensity columns
- [x] **IMPORT-03**: System auto-suggests likely protein ID and intensity columns based on naming patterns and data types
- [x] **IMPORT-04**: User sees a data preview (first rows) during column mapping
- [x] **IMPORT-05**: User can optionally log2-transform intensities on import
- [x] **IMPORT-06**: Imported generic matrix receives same semantic types as MaxQuant import, enabling full downstream pipeline

### QC Dashboard

- [x] **QC-01**: User can open a QC dashboard showing all QC viewers in a linked layout
- [x] **QC-02**: User can view an MA plot (M vs A) showing intensity-dependent bias between conditions
- [x] **QC-03**: User can view a missing values heatmap showing presence/absence pattern across samples
- [x] **QC-04**: User can view a sample correlation matrix as a heatmap of pairwise Pearson correlations
- [x] **QC-05**: User can view per-sample intensity distributions (box plots)
- [x] **QC-06**: User can view a CV plot showing measurement variability per protein within replicates
