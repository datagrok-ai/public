# Media

`Media` is a [package](https://datagrok.ai/help/develop/develop#packages) for the [Datagrok](https://datagrok.ai)
platform that previews audio and video files in the file browser using the browser's built-in HTML5
`<audio>` and `<video>` elements — no third-party playback libraries.

## Supported formats

* **Audio**: `mp3`, `wav`, `flac`, `ogg`, `oga`, `opus`, `m4a`, `aac`, `weba`
* **Video**: `mp4`, `m4v`, `webm`, `ogv`, `mov`

Actual playback depends on the codecs the user's browser supports. Controls (play/pause, seek,
volume, fullscreen for video) are provided natively.
