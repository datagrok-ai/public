Box plot: split · covariate (ratio / regress-out / ANCOVA) · method

One interface for both datasets. A covariate can be handled three mutually exclusive ways: Ratio and Regress-out transform Y (then a normal test), while ANCOVA takes the covariate into the test itself — so selecting ANCOVA disables the Ratio/Regress-out modes (otherwise it would double-correct).
