You may include a script into an external web page. Datagrok supports embedding 
via the [iframe](https://www.w3schools.com/tags/tag_iframe.ASP) element.

To embed a script:
- click on the hamburger icon in the top left corner
- choose `Embed...`

You will see a dialog with the HTML code on top that you can use to embed this into an external site.
Note that you can change parameter values right in the URL.
