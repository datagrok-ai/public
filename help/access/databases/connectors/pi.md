---
title: "PI"
---

Provides access to the
[PI](https://www.aveva.com/en/products/aveva-pi-system/) database using SQL queries via a JDBC driver.

## Connection parameters

````json
{
   "parameters": {
    "accessServer": "",
    "port": port,
    "db": "db-name",
    "cacheSchema": false,
    "cacheResults": true,
    "ssl": false,
    "connString": ""
  }
}
````
