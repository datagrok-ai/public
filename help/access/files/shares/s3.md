Provides access to [S3](https://aws.amazon.com/s3/) as a [file share](../files.md).

## Connection parameters

```json
{
  "parameters": {
    "region": "us-east-1",
    "bucket": "",
    "dir": "",
    "encryption": ""
  },
  "credentials": {
    "parameters": {
      "accessKey": "",
      "secretKey": ""
    }
  }
}
```
