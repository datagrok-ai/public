# MongoDB

Provides access to [Apache HBase](https://hbase.apache.org/) NoSQL database
using SQL queries via JDBC driver.

## Connection parameters

```json
{
    "server": "",
    "port": "",
    "db": "",
    "connString": ""
}
```
