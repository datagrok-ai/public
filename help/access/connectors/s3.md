# S3

Provides access to [S3](https://aws.amazon.com/s3/) as [file share](files.md).

## Connection Parameters

```json
{
  "Region": "us-east-1",
  "Bucket": "",
  "Dir": "",
  "Encryption": "",
  "credentials": {
    "parameters": {
      "AccessKey": "",
      "SecretKey": ""
    }
  },
}
```
