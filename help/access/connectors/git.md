# Git

Provides access to [Git](https://git-scm.com/) repository as [file share](files.md).
Works with any Git repository.

## Connection Parameters

```json
{
  "url": "https://github.com/datagrok-ai/public.git",
  "dir": "packages/ApiSamples",
  "indexFiles": true,
  "privateRepo": false
}
```
